This example page uses the `hidden-home` layout to put a calm full-screen cover in front of the rest of your site while you work on changes. Add a file named `static/images/hidden-home-cover.webp` in your site to replace the default background image.

Use it as a standalone maintenance URL while you prepare a release, or temporarily switch your homepage to the same layout if you need the entire site to show a maintenance message.

Since there are no navigation links on this page, use your browser back button or return to the [example site](../).
